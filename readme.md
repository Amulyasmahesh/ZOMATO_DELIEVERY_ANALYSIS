# 🍽️ Zomato Delivery Data Analysis

## 📌 Overview
This project analyzes Zomato restaurant data to extract insights about customer preferences, ratings, and pricing trends.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Restaurants offering online delivery have higher average ratings
- Mid-range pricing correlates with better ratings
- Most restaurants fall in the 3.5–4.0 rating range

## 📈 Visualizations
- Rating distribution
- Cost vs Rating
- Online Order vs Rating
- Correlation heatmap

## 🚀 Outcome
Derived actionable insights that can help food delivery platforms improve customer experience and optimize pricing strategies.
